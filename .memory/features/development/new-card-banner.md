# Memory: features/development/new-card-banner
Updated: now

When a new card is opened by the receiving team (e.g., China opens a card created by Brazil), a 'New Request' banner appears in the timeline showing the card's title, description/desired outcome, and image thumbnail. Action buttons for Comment, Ask a Question, Snooze, and Upload are included for immediate interaction. This banner displays only when `is_new_for_other_team` is true and there are no higher-priority banners (questions, samples, etc.).
